Índice de estrés externo para Chile

Indicador compuesto de condiciones externas relevantes para una economía pequeña y abierta.

Motivación

Chile es una economía pequeña y abierta, expuesta a cambios en condiciones financieras globales, precios de commodities y apetito por riesgo internacional.

Este proyecto construye un índice sintético de estrés externo a partir de variables como VIX, precio del cobre, petróleo, tasas de Estados Unidos y dólar global.

Esta versión inicial usa datos simulados solo para validar la arquitectura visual del sitio. La siguiente etapa es reemplazar la simulación por datos reales desde FRED, BCCh u otra fuente pública.

Metodología

El índice se construye transformando las series a frecuencia mensual, estandarizándolas mediante z-scores, ajustando signos según interpretación económica y agregando por promedio simple o PCA.

Visualización interactiva inicial

Interpretación

Valores positivos indican un entorno externo más restrictivo o adverso para Chile. Valores negativos indican condiciones externas más favorables.
