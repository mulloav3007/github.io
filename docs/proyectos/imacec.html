Pregunta económica

El IMACEC es una de las señales principales para evaluar la coyuntura económica chilena. Este proyecto busca responder una pregunta práctica:

¿Es posible anticipar la evolución reciente del IMACEC total y no minero usando información mensual disponible antes de la publicación oficial?

El objetivo no es producir una proyección oficial, sino construir una herramienta reproducible de seguimiento coyuntural que combine datos macroeconómicos, variables calendario, indicadores sectoriales y modelos econométricos simples.

Datos utilizados

El pipeline integra información mensual de actividad, comercio, crédito, precios, mercado laboral e indicadores sectoriales. La versión actual considera:

  • IMACEC total e IMACEC no minero.
  • Ventas minoristas.
  • Flujo y número de operaciones de crédito.
  • UF e inflación implícita asociada.
  • Desempleo.
  • Cobre y petróleo.
  • Días hábiles y variables calendario.
  • Indicadores sectoriales INE: minería, manufactura, comercio y electricidad.

Enfoque metodológico

La versión actual implementa modelos lineales de nowcasting con rezagos de actividad, indicadores contemporáneos, dummies mensuales, efectos calendario y controles puntuales para episodios atípicos. También incluye una especificación alternativa con indicadores sectoriales INE.

El flujo de trabajo está organizado en módulos:

  1. Descarga de series desde la API del Banco Central.
  2. Transformación mensual y construcción de variaciones interanuales.
  3. Integración de calendario de días hábiles.
  4. Estimación de modelos para IMACEC total y no minero.
  5. Construcción del mes objetivo.
  6. Exportación de tablas, gráficos y archivos procesados.

# Detectar automáticamente la raíz del repositorio Economics

find_project_root <- function(path = getwd()) {
  path <- normalizePath(path, winslash = "/", mustWork = TRUE)
  
  for (i in seq_len(10)) {
    if (file.exists(file.path(path, "_quarto.yml"))) {
      return(path)
    }
    
    parent <- dirname(path)
    
    if (identical(parent, path)) {
      break
    }
    
    path <- parent
  }
  
  stop("No pude encontrar la raíz del proyecto. Revisa que exista _quarto.yml.")
}

project_root <- find_project_root()

# Hace que los chunks siguientes trabajen desde la raíz del proyecto
knitr::opts_knit$set(root.dir = project_root)

# Helper para construir rutas robustas
path_project <- function(...) {
  file.path(project_root, ...)
}

# -------------------------------------------------------------------
# Elegir archivos reales si existen; si no, usar archivos demo
# -------------------------------------------------------------------

choose_file <- function(real_path, demo_path) {
  real_full <- path_project(real_path)
  demo_full <- path_project(demo_path)
  
  if (file.exists(real_full)) {
    list(path = real_full, status = "real")
  } else if (file.exists(demo_full)) {
    list(path = demo_full, status = "demo")
  } else {
    stop(
      "No existe ni el archivo real ni el archivo demo:\n",
      "- Real: ", real_full, "\n",
      "- Demo: ", demo_full
    )
  }
}

history_file <- choose_file(

## IMACEC total: observado versus ajuste/nowcast

```{r plot-total}
hist_plot <- hist |>
  filter(Periodo >= max(Periodo, na.rm = TRUE) %m-% months(95))

g_total <- ggplot(hist_plot, aes(x = Periodo)) +
  geom_hline(yintercept = 0, linewidth = 0.25, color = "grey70") +
  geom_line(aes(y = imacec, color = "Observado"), linewidth = 0.75, na.rm = TRUE) +
  geom_line(aes(y = imacec_fit, color = "Ajuste / nowcast"), linewidth = 0.75, linetype = "dashed", na.rm = TRUE) +
  geom_point(
    data = filter(hist_plot, tipo == "Nowcast"),
    aes(y = imacec_fit, color = "Nowcast"),
    size = 2.5,
    na.rm = TRUE
  ) +
  scale_color_manual(values = c("Observado" = "#1f4e79", "Ajuste / nowcast" = "#b03a2e", "Nowcast" = "#7f1d1d")) +
  labs(
    x = NULL,
    y = "Var. 12m (%)",
    color = NULL,
    title = "IMACEC total: observado versus ajuste/nowcast"
  ) +
  theme_minimal(base_size = 12) +
  theme(legend.position = "top", panel.grid.minor = element_blank())

plotly::ggplotly(g_total)
```
